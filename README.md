# QUICK START

Strategy 1: Bet by previous consecutive result

- Example:

  1. If result of 3 previous round = DOWN and current LIVE ROUND = down => bet UP
  2. If result of 3 previous round = UP and current LIVE ROUND = UP => bet DOWN
